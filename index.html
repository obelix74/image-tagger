üö¶ Getting Started

Prerequisites

  • Node.js 18+
  • Google Gemini API key (for cloud AI) OR Ollama installation (for local AI)

Quick Setup

# Clone the repository
git clone <repository-url>
cd image-tagger

# Run setup script
./scripts/setup.sh    # Unix/Linux/Mac
# or
scripts\setup.bat     # Windows

# Start the application
./scripts/start.sh    # Unix/Linux/Mac
# or  
scripts\start.bat     # Windows

Access the Application

After setup, open your browser and navigate to http://localhost:5173
